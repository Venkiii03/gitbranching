**Level one : Cherry-pick intro** <br>
This image demonstrates git cherry-pick, which allows you to copy specific commits from different branches and apply them to the current branch.
Only the selected commits are moved, without merging the entire branch history. <br>

**Command used:** <br>
```git cherry-pick C3 C4 C7```

**Level two : Interactive Rebase Intro** <br>
This image introduces interactive rebase, which allows you to rewrite, reorder, or modify commits before applying them to another base.
It helps clean up commit history before pushing changes. <br>

**Command used:** <br>
```git rebase -i HEAD~4```
